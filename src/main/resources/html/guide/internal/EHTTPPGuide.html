HTTP Proxy extension allows you to transmit data over HTTP proxies like ZAP and Burp.

Tags

In order to specify, which PDUs should go through proxy, you can use the following tags (all PDUs are sent over HTTP proxy by default):

  • no_httpp: tagged PDU won't be transfered over HTTP proxy.
  • httpp: tagged PDU will be transfered over HTTP proxy even if it already has no_httpp tag.
