Extensibility

One of core properties of PETEP is its extensibility. Extensions can be written in Java using PetepLib.jar library - for more information about extension development, please check out PETEP website using your web browser.

Proxy extensions

Proxy extensions allow you to implement your own proxies for other protocols (TCP, UDP, SMTP, ...). Data from these protocols will be transmitted through the PETEP core and its modules.

Internal proxy extensions:

  • TCP proxy: simple TCP proxy.
  • HTTP proxy: very simple HTTP proxy with basic WebSockets support.
  • Test proxy: for testing purposes.

Intercept extensions

Intercept extensions allow you to implement your own intercepting module which can modify PDUs, create GUI tabs etc.

Internal intercept extensions:

  • Connection View: viewer of currently opened connections.
  • External HTTP Proxy: transmission of PDUs through HTTP proxy (e.g. ZAP, Burp).
  • Logger: logs PDUs to plain-text files.
  • Modifier: modification rules for automatic data replacements etc.
  • Tagger: tag rules for automatic PDUs tagging.
  • Catcher: manual PDU intercept editor.
